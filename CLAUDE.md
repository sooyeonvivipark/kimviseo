# 김비서 — 업무 대시보드

현대적인 글래스모피즘 디자인의 비서 업무 관리 대시보드.

## 🎯 주요 기능

- **할 일 목록** — 우선순위 색상 표시, 체크박스로 완료 여부 관리
- **이번 주 일정** — 월~금 요일별 일정 표 조회
- **프로젝트 진행률** — 6개 프로젝트의 진행 상황을 프로그레스 바로 시각화
- **매출 요약** — 총 매출, 거래건수, 평균단가, 최다판매 제품

## 📁 파일 구조

```
/
├── dashboard.html          # 메인 대시보드 (HTML, CSS, JavaScript 통합)
├── CLAUDE.md              # 프로젝트 문서
└── 김비서-데이터/         # 데이터 폴더
    ├── 업무목록.csv       # 할 일 목록 데이터
    ├── 주간일정.txt       # 주간 스케줄
    ├── 프로젝트현황.csv   # 프로젝트 진행률 데이터
    └── 매출데이터.csv     # 매출 통계 데이터
```

## 🛠️ 사용 기술

- **HTML5** — 시맨틱 마크업
- **CSS3** — 글래스모피즘, 그라디언트, 애니메이션
  - Backdrop-filter (frosted glass 효과)
  - Linear & Radial Gradient
  - Shimmer 애니메이션
- **JavaScript (Vanilla)** — 다크/라이트 모드 토글, localStorage 저장

## ✨ 특징

- 🌙 **다크/라이트 모드** — 자동 저장 (localStorage)
- 💎 **프리미엄 디자인** — 글래스모피즘 + 부드러운 그림자
- 📱 **반응형 레이아웃** — 모바일, 태블릿, 데스크톱 대응
- ⚡ **부드러운 애니메이션** — Cubic-bezier 트랜지션
